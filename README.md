# discord_bot
게임 도움용 디스코드 봇 

## 계획 
- ### 게임 전직 조회 (롤,롤토체스,발로란트,배그,오버워치)
  - 추가 기능
    - 롤: 나만의 상점,
    - 롤토체스 : 조합 보기 
    - 발로란트 : 오늘의 상점,야시장
    - 배그 : 이벤트? 
    - 오버워치 : (뭐있더라)
- ### chatGPT 를 이용한 게임 도음
  - 롤
    - 챔피언 강의 유튜브 추천 
    - 유즘 나오고 있는 메타   
    - 새로운 맵( 예: urf)
  - 롤토체스
    - 전직 결과에 대한 피드백 
  - 발로란트
    - 발로란트 맵별로 추천 가능
  - 공통
